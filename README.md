## 📌 About This Repository

This repository is a personal learning collection of ebooks, notes, and references related to **Artificial Intelligence**, **Machine Learning**, **Deep Learning**, **Natural Language Processing**, **Computer Vision**, and **MLOps**.

The purpose of this repository is to organize learning materials in one place so that students, researchers, and AI learners can quickly find useful references for studying and building AI projects.

## 🗺️ Suggested Learning Path

If you are new to AI, you can follow this learning order:

### 1. Python & Data Science Basics

Start with:

* Python fundamentals
* NumPy
* Pandas
* Matplotlib
* Basic statistics

Recommended topics:

```text
Python → NumPy → Pandas → Data Visualization → Statistics
```

---

### 2. Machine Learning Foundations

After Python basics, study:

* Supervised Learning
* Unsupervised Learning
* Regression
* Classification
* Clustering
* Model Evaluation

Recommended books:

```text
Introduction to Machine Learning with Python
Hands-On Machine Learning with Scikit-Learn
```

---

### 3. Deep Learning

Then move to:

* Neural Networks
* Backpropagation
* CNN
* RNN
* LSTM
* Transformers

Recommended books:

```text
Deep Learning by Ian Goodfellow
Fundamentals of Deep Learning
Hands-On Machine Learning with PyTorch
```

---

### 4. Natural Language Processing

For NLP and LLM topics:

* Text preprocessing
* Tokenization
* Word embeddings
* Transformers
* BERT
* Large Language Models

Recommended books:

```text
Natural Language Processing with Python
NLP with Transformer Models
Foundations of Large Language Models
Hands-On Large Language Models
```

---

### 5. Computer Vision

For image-based AI:

* Image processing
* CNN
* Object detection
* Vision Transformers

Recommended books:

```text
Computer Vision Algorithms and Applications
Hands-On Computer Vision with Transformers
```

---

### 6. MLOps & AI Engineering

For real-world AI deployment:

* Model serving
* Model monitoring
* CI/CD for ML
* AI system design
* Foundation model applications

Recommended books:

```text
Practical MLOps
AI Engineering
Designing Machine Learning Systems
```

## 🛠️ How to Clone This Repository

Because this repository may contain large PDF files, it is recommended to use **Git LFS**.

### Step 1: Install Git LFS

```bash
git lfs install
```

### Step 2: Clone the repository

```bash
git clone https://github.com/Xenko2005/EBook for AI.git
```

### Step 3: Pull large files

```bash
git lfs pull
```

Replace:

```text
YOUR_USERNAME
YOUR_REPOSITORY_NAME
```

with your actual GitHub username and repository name.

## ⭐ Recommended Study Strategy

To study effectively:

1. Do not try to read every book from start to finish.
2. Choose one main topic at a time.
3. Take notes while reading.
4. Implement algorithms with Python.
5. Build small projects after each topic.
6. Review important chapters regularly.
7. Combine theory with practice.
